# Snake & Ladder (CLI)

Run the game with Python 3:

```powershell
python snake_and_ladder.py
```

Controls:
- Enter number of players (1-4).
- Enter a name for each player, or leave empty to make a computer player.
- On your turn press Enter to roll. Type `q` to quit.

Rules implemented:
- Standard 1-100 board.
- Must roll exact number to reach 100.
- Snakes and ladders are applied automatically.
